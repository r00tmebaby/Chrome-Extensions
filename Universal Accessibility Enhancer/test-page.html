Test Instructions

Open the UAE extension popup (click the extension icon) and try these features:

  1. Easy Read Mode: Should narrow the content width and dim the sidebar/header/footer
  2. Focus Mode: Should blur non-main content
  3. Font Changes: Try OpenDyslexic or Atkinson Hyperlegible fonts
  4. Vision Boost: Should increase contrast and saturation slightly
  5. High Contrast: Should apply white text on black background
  6. Color Filters: Try Protanopia, Deuteranopia, or Grayscale
  7. Reduce Motion: Should stop the red box animation below
  8. Text Size: Adjust in options to see text scale
  9. Reader View: Click the Reader View button in the popup
