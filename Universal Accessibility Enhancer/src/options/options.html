üöÄ Getting Started

What is UAE?

Universal Accessibility Enhancer (UAE) makes websites more accessible by adjusting text, colors, layout, and behavior to meet your specific needs. All changes happen locally in your browser - no data is collected or sent anywhere.

Quick Start Guide

  1. Try a Preset: Click Dyslexia, ADHD, or Low Vision buttons in the Modes tab for instant configuration
  2. Adjust Settings: Fine-tune individual options across different tabs
  3. See Changes Live: All adjustments apply immediately to the current page as a preview
  4. Save Your Preferences: Click "Save Global" to apply everywhere, or "Save Domain Override" for specific sites

Who Is This For?

  • People with dyslexia or reading difficulties
  • People with ADHD or focus challenges
  • People with visual impairments
  • People with color blindness
  • People with photosensitivity or epilepsy
  • Anyone who wants a better reading experience!

‚ö° Quick Presets

Dyslexia Preset

Best for: People with dyslexia or reading difficulties

What it does:

  • Activates Easy Read mode (simplified layout, increased spacing)
  • Switches to OpenDyslexic font (designed specifically for dyslexia)
  • Increases text size to 110% and line height to 2.0
  • Reduces motion (stops distracting animations)
  • Dims non-essential content

ADHD Preset

Best for: People with ADHD or focus difficulties

What it does:

  • Activates Focus Mode (blurs distractions, highlights main content)
  • Reduces motion (removes all animations and transitions)
  • Auto-pauses videos and media
  • Uses System UI font for familiarity
  • Increases line height to 1.8 for better readability

Low Vision Preset

Best for: People with low vision or visual impairments

What it does:

  • Activates Easy Read mode
  • Increases text size to 140% and line height to 2.2
  • Enables Vision Boost (increases contrast and saturation)
  • Activates High Contrast mode (white text on black background)
  • Uses Verdana font (highly legible)

üìñ Modes & Layout

Easy Read Mode

Simplifies web pages for easier reading by:

  • Narrowing line length to 70-72 characters (optimal for reading)
  • Increasing spacing between paragraphs
  • Dimming sidebars, headers, and footers (opacity reduced to 25%)
  • Hiding sticky headers and floating ads
  • Centering main content automatically

Focus Mode

Helps you concentrate by:

  • Blurring everything except main content (4px blur + 40% brightness reduction)
  • Highlighting the main article/content area
  • Removing distractions without hiding them completely
  • Works great for reading articles or long-form content

Font Families

OpenDyslexic

Specially designed for dyslexia with weighted bottom letters to prevent letter confusion.

Atkinson Hyperlegible

Created by Braille Institute for maximum legibility - distinguishes similar letters clearly.

System UI

Your device's native font - familiar and fast-rendering.

Verdana / Georgia / Arial

Readable standard fonts optimized for screen reading.

Large (size only)

Keeps current font but increases size to 110%.

üëÅÔ∏è Vision Adjustments

Vision Boost

Gently increases contrast (1.2x) and saturation (1.05x) for better visibility without being harsh.

Great for:

  • Low contrast websites
  • Viewing in bright sunlight
  • Mild vision impairments
  • Reducing eye strain

High Contrast Mode

Maximum contrast for severe vision impairments:

  • Forces white text on black background
  • High contrast borders on all elements
  • Removes background images
  • Simplifies color schemes drastically

Color Filters

Simulates color transformations for different types of color blindness:

Protanopia

Red-blind - Helps distinguish red from other colors

Deuteranopia

Green-blind - Transforms greens to be more visible

Tritanopia

Blue-blind - Adjusts blue tones for better distinction

Grayscale (Achromatopsia)

Complete color blindness - Removes all colors

Larger Hit Targets

Increases the size of interactive elements:

  • All buttons, links, and inputs get minimum 36x36px size
  • Increased padding (8px-12px) for easier clicking
  • Helpful for motor impairments or touch screen users

üìè Reading Comfort

Text Size Slider (80% - 200%)

Scales all text on the page proportionally:

  • 80-90%: Smaller text for fitting more on screen
  • 100%: Default website size
  • 110-140%: Comfortable reading size for most users
  • 140-200%: Large text for low vision users

Changes apply in real-time as you drag the slider!

Line Height Slider (1.2 - 2.6)

Adjusts spacing between lines of text:

  • 1.2-1.4: Compact spacing
  • 1.6-1.8: Comfortable reading (recommended)
  • 2.0-2.6: Extra spacing for dyslexia or visual tracking difficulties

Higher values prevent lines from blending together.

Reduce Motion

Removes all animations and transitions:

  • Stops spinning loaders, sliding banners, bouncing elements
  • Prevents motion sickness and distraction
  • Essential for vestibular disorders or photosensitivity
  • Makes pages feel faster and more stable

Keyboard Focus Boost

Enhances the visual outline when navigating with Tab key:

  • 3px thick yellow outline (vs standard thin outline)
  • 3px offset for better visibility
  • Helps keyboard-only users see where they are

Underline Links

Forces underlines on all hyperlinks:

  • Makes links immediately identifiable
  • Doesn't rely on color alone
  • Helpful for color blindness

üé¨ Media & Speech

Auto Pause Media

Automatically pauses autoplaying videos and audio:

  • Stops videos from playing automatically
  • Pauses background music
  • You can still click to play manually
  • Freezes animated GIFs (converts to static PNG)
  • Reduces sensory overload and distractions

Read Aloud (Text-to-Speech)

Reads main content using browser's built-in voice:

  • Finds main article content automatically
  • Reads headings, paragraphs, and list items
  • Up to 8,000 characters per reading
  • Uses native browser TTS (no external services)
  • Toggle off to stop reading immediately

Note: Voice quality depends on your browser and OS.

  • Dyslexia: Reading difficulties
  • ADHD / Focus difficulty: Attention challenges

Save Options

Save Global

Apply these settings to all websites you visit. Changes persist across browser restarts.

Save Domain Override

Save settings only for the current website (e.g., different settings for Wikipedia vs Reddit). Domain-specific settings take priority over global settings.

Reset to Defaults

Restore all settings to safe default values. This clears all customizations.

Preview vs Save

Important: All changes show as a live preview on the current page, but they're temporary until you click Save!

  • Adjust sliders/checkboxes ‚Üí See changes immediately
  • Click "Save Global" ‚Üí Make changes permanent
  • Close options without saving ‚Üí Changes are lost

Quick Access Shortcuts

Use these keyboard combinations for instant access:

  • Alt + Shift + F - Toggle Focus Mode on/off
  • Alt + Shift + E - Toggle Easy Read Mode on/off
  • Alt + Shift + V - Toggle Vision Boost on/off

Works on any page without opening the popup or options!

Customizing Shortcuts

Chrome

  1. Go to chrome://extensions/shortcuts
  2. Find "Universal Accessibility Enhancer"
  3. Click the pencil icon next to each shortcut
  4. Set your preferred key combinations

Firefox

Keyboard shortcuts are not customizable in Firefox currently. The default shortcuts will work as listed above.

üîß Troubleshooting

Extension Not Working?

Reload the page

Some sites need a refresh after installing the extension or changing settings.

Check restricted pages

Extension can't work on:

  • chrome://extensions or about:debugging
  • Browser settings pages
  • Chrome Web Store or Firefox Add-ons pages

Try incognito mode

Check if other extensions are conflicting. Enable UAE in incognito mode from the extensions page.

Clear settings

Click "Reset to Defaults" in Advanced tab to restore default settings.

Changes Not Saving?

  • Make sure to click "Save Global" or "Save Domain Override"
  • Preview changes are temporary - you must explicitly save them
  • Check browser permissions are granted
  • Try reloading the extension from the extensions page

Fonts Not Loading?

  • OpenDyslexic and Atkinson fonts are embedded in the extension
  • If fonts don't show, try reloading the extension
  • Some websites may override fonts with !important CSS
  • Check browser console for errors (F12)

Performance Issues?

  • Disable Read Aloud (TTS) if not needed
  • Use fewer features at once
  • High Contrast mode is performance-intensive on complex sites
  • Try disabling other browser extensions temporarily

Still Having Issues?

If problems persist:

  1. Check the browser console for errors (F12 ‚Üí Console tab)
  2. Try removing and reinstalling the extension
  3. Verify you're using a supported browser (Chrome, Firefox, Edge, Brave)
  4. Check if the website has unusual security policies
